# Spider_netbian

#### 介绍
彼岸图网图片爬虫： 
https://pic.netbian.com/


#### 安装教程

1.  pip安装scrapy框架
2.  pip安装PIL图片处理库
3.  运行startup.py文件 或在项目文件夹下运行 scrapy crawl netbianSpider

#### 使用说明

1.  首先提示爬取图片专区 有12个专区可以选择，输入对应数字即可
2.  输入起始页和终止页 均为数字
3.  提示开始下载...
